Oberon 2.3.8 for Linux

You must download both the install script and tar file. Note that this version may not function properly under Redhat 9.0 or later due to incompatibilities with some of the C libraries. The system has been tested under Mandrak 9.1 and 9.2. Follow the instructions below to install Oberon on your own system:
  • Login as "root" or change to super-user mode by using the su command.
  • Change the file permissions of the installation script by executig the command chmod 744 inst.oberon
  • Create a directory for oberon, for example mkdir /oberon. This directory is used to store the oberon installation and can be shared between different users on the same system.
  • Execute the command ./inst.oberon x86_Linux_Oberon_r238a.Sys.tgz /oberon /usr/local/bin/oberon
  • Create a working directory under your normal user login on the system, for example mkdir ~/oberon. You can now execute the command oberon from this directory and all your files will be stored here.

Implementation of the Banker's Algorithm

You can download the implementation of the Banker's algorithm here. The implementation is based on the examples in your text books. The various commands that can be executed are listed at the end of the source code.

Update for osExceptions

The updated version for osExceptions contains additional code to extract the trap codes generated by the OP2 compiler as part of its runtime error checking and will allow your system to identify the following runtime errors:
  • RETURN missing
  • Missing case in CASE statement
  • ASSERT failure
  • Index out of range
