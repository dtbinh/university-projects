RW778 -- Advanced Linux and Awk

11 April 2006: Exercises

  1. Study the gawk manual provided.
  2. Consider the data file genomedata.txt. Search for all occurrences of the pattern GGTTAA. Note that these lines occur as six separate lines of the form ... SY="G", and so on. Now display the DE line of the groups in which the pattern GGTTAA does occur.
  3. Consider the data file testdata.tar.gz. Unpack it in a new directory with the command tar -xzvf testdata.tar.gz. This should create 65000 data files. Please don't try to ls -al, but a straight ls should work. Now write combined shell and awk scripts to
    1. Replace all occurrences of the word start with the word START.
    2. Count the number of occurrences of the word FINAL .
    3. Count the number of files which have less than 5 lines in their descriptions.
    4. Reorder each of the test files so that the lines containing the word FINAL occur directly below the line containing the word START.
